8 Data wrangling

In this chapter we begin to delve into manipulation of data in the form of a data frame or tibble. In so doing, we will introduce the tidyverse package and the various verbs (function) it provides.

The tidyverse package is not just a single package but a composit of a group of packages. There include among others the dplyr package. Most of the function we will be employing in this chapter comes from dplyr.

We begin by reading in the blood_donors.xls

df_blood <- readxl::read_xls("blood_donors_1.xls")

Below we rename the variables hb to hemog and id to studyid using he rename function, and then show the first 5 records with the head function.

8.3 Subsetting data

In this subsection we demonstrate the use of the filter and select function to select specific records and variables in a tibble. Below we filter to select all records with hb > 12g/dl and the keep only the id, hb and sex columns.

df_blood %>% 
    filter(hb > 12) %>% 
    select(id, hb, sex)

8.4 Generating new variables
